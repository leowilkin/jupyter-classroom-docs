This guide will provide you the instructions and commands nessesary to run Jupyter Classroom, and by extension, JupyterHub. 

**This gets technical**. If you have not encountered running terminal commands, or managing infrastructure before, I recommend you ask an IT professional at your school/institution for support.

For simplicity, I'll be using [TLJH](https://tljh.jupyter.org/en/latest/index.html) (The Littlest JupyterHub). If you're expecting to run with over 100 users on one instance, please read [this document](https://tljh.jupyter.org/en/latest/topic/whentouse.html#) from Jupyter.org.

Finally, you'll need a compute instance. This instruction guide will walk you through running this on-prem, but if you're not up for that, here's a few useful guides to running JupyterHub on a different VPS solution.
